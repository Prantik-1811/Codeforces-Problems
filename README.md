# Codeforces C++ Solutions

This repository contains C++ source files for Codeforces problems written in C++.
Each solution file is named by problem ID and letter, for example `2133A.cpp`.

## Build

Compile a source file with a C++ compiler, for example:

    g++ -std=c++17 -O2 -Wall -Wextra -o 2133A.exe 2133A.cpp

## Notes

- Compiled executables are ignored by `.gitignore`.
- If `.exe` files are already tracked in Git, remove them from the index with `git rm --cached *.exe` and commit the change.
